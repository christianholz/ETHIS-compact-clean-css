# ETHIS-clean-css

Transforms the ETH ETHIS (SAP Fiori Launchpad) tile UI into a compact text list

![Compact view](eth_ethis_compact.png)

## Usage

load `ethis.css` via a Chrome user CSS extension (e.g. Stylus) for `ethis.ethz.ch/sap/bc/ui2/flp*`.

Or load `chrome-ethis-compact` as extension in [chrome://extensions]().
